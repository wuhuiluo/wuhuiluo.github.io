面试准备

箭头函数和普通函数的区别

  1. 箭头函数和普通函数的样式不同，箭头函数的语法更加简洁，箭头函数是 => 定义函数，而普通函数是function定义函数

  2. 箭头函数会捕获其上下文的this值，作为自己的this值，定义的时候就确定并固定了

  3. 箭头函数并不能作为构造函数使用，也不能使用new关键字 原因：因为箭头函数没有自己的this值，它的this值继承了外层执行环境中的this，且this指向永远不变，作为构造函数其的this要是指向创建的新对象

  4. 箭头函数没有自己的arguments，在箭头函数中访问arguments实际上获得的是外层函数执行环境中的值

  5. call、apply、bind不会影响其this的指向

  6. 箭头函数没有原型prototype

  7. 箭头阿黄念书不能当作Generator函数，不能使用 yield关键字

var、let、const之间的区别

从以下三个方面

  1. 变量提升方面：var声明的变量存在变量提升，即变量可以在声明前就使用，值为undefined

let和const不存在变量提升问题，其实是有提升的，只不过是let和const具有一个暂时性死区，即没有看到其赋值时，就不能使用、他们所声明的变量一定要在声明后使用，否则报错

  1. 块级作用域方面

var不存在块级作用域，let和const存在块级作用域

声明方面

var允许重复声明变量，let和const在同一作用域不允许重复声明变量，其中const声明一个只读的常量（因为如此，其声明就一定要赋值，不然就会报错），一旦声明，常量的值就不能改变

如何使const声明的对象内属性不可变，只可读呢

如果const声明了一个对象，对象里的属性是可以改变的

因为const声明的obj只是保存着其对象的引用地址，只要地址不变，就不会出错

使用Object.freeze(obj)冻结boj，就能使其内的属性不可变，但他有局限，就是obj对象中要是有属性是对象，该对象内的属性还能改变，要全不可变，就需要使用递归的方式一层一层的全部冻结

什么是暂时性死区

记住ES6中的一个特性，如果去区块中存在let和const命令声明的变量，从一开始就形成了封闭作用域，因为JS清楚的感知到了name是用let声明在当前这个代码快内的，所以会给这变量加上暂时性死去的限制，它就不往外探出头了

什么是块级作用域

在ES6中新出的两个关键字 let和 const中，自带块级作用域，块级作用域相当于是只在这块代码中生效，如果它被大括号{}所包围，那么大括号中就是一段代码快，代码块中使用let和const声明的变量也被成为局部变量

基本数据类型和引用数据类型的区别

基本数据类型: Number String Boolean Null Undefined Symbol BigInt

  1. 基本数据类型的值是不可改变的（重新赋值属于改变属性名的指向了，而不是对值进行操作），
  2. 基本数据类型不可以添加属性和方法
  3. 基本数据类型的赋值是简单赋值，基本数据类型的比较是值的比较
  4. 基本数据类型是存放再栈区的

引用数据类型: Function Array RegExp Date

  1. 引用类型的值是可以改变的，例如对象就可以通过修改对象属性的值更改对象
  2. 引用类型可以添加属性和方法
  3. 引用类型的赋值是对象的引用，存储的只是对象的指针地址
  4. 引用类型打包i叫是引用（指针地址）的比较
  5. 引用类型是同时保存在栈区和堆区中的，栈区保存变量标识符和指向堆内存的地址

new的过程中发生了什么

  1. 创建一个空对象
  2. 该对象的隐式原型指向该构造函数的原型
  3. 构造函数内部的this被赋值给这个新对象
  4. 执行构造函数内部的代码(给新对象添加属性)
  5. 如果函数没有返回其他对象，那么new表达式中的函数调用会自动返回则会个新对象
